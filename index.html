迪迦奥特曼

       奥特曼战士，在三千万年前的超古代时期就出现在地球的光之巨人，并非是地球出生的奥特曼。
在超古代时期，人类因齐杰拉花朵的花粉所迷惑选择了灭亡后，光之巨人无法干涉人类的选择，所以迪迦和另外两个巨人将自己的身体隐藏在巨大的金字塔中，还原成光的形态离开了地球。最终邪神卡达诺佐亚带着毁灭的黑暗降临，将人类超古代文明彻底毁灭。
直到三千万年后（2007年），当其他两座巨人石像被哥尔赞及美尔巴摧毁后，剩下的一尊因大梧化成光进入石像，而令迪迦的躯体成功复活，此后迪迦一直以圆大梧的身份在地球上活跃。其名字“迪迦（TIGA）”一词，意为印度尼西亚语中神圣的数字"3"。

  • 变身者：圆大梧、阿姆伊、圆翼（后两者均为暂时）
  • 身高：53米（基本型态）/ 120米（闪耀型态）
  • 体重：44000吨（基本型态）/ 100000吨（闪耀型态）
  • 年龄：3000万岁以上
  • 活动时间：3分钟
  • 特别能力：可瞬间移动，随着距离增加能量消耗，距离上限即为能量上限。

变身器

  1. 神光棒（スパークレンス）

    2. 能使圆大梧变身成迪迦的道具，在第1话大梧和迪迦石像合为一体并打败怪兽后，出现在大梧队服的左胸袋中。
    有着水晶及大理石般的材质，其上端有着和迪迦胸部防护甲相似的部分，是变身时发光的镜片覆盖的部分。
    通常变身之时会逆时针旋转神光棒，手再举向天空、 随即神光棒顶部的尖端会左右展开，放射出绚丽的光芒将大梧包围，完成变身。
    根据场合不同会省略姿势，曾被正木敬吾夺走用光粒子转化器来编入自己的基因。

  2. 黑暗神光棒（ブラックスパークレンス）

    3. 在电影版《最终圣战》首次登场，是迪迦3000万年前的恋人卡蜜拉所赐予，外型和神光棒一样，然而整体颜色为黑色和灰色。
    蕴藏着和神光棒相反的“黑暗力量”，使用此变身器会变身为黑暗迪迦（ティガダーク）。

  3. 青铜神光棒（青铜のスパークレンス）

    4. 在外传《迪迦·奥特曼-远古复苏的的巨人》登场，青铜材质的神光棒。
    主体大部分呈绿色，花纹为鲜红色，且造型上比火花棱镜要大，剧中称呼为“远古的神器”（远古の神器）。

型态变化

  1. 复合型（マルチタイプ）

    2. 迪迦的基本型态，身体线条为红紫金银四色，善于与各种光线技术和武术格斗的战士。
    拥有平衡的力量和速度。能使出不同光子射线，如威力极强，两腕在前方交叉后，向左右展开收集能量，再组合成L字型放出的哉佩利敖光线（ゼペリオン光线）。
    虽然能转变型态，但综合来说多重型态在敏捷、强力相比是较为全方位的形态，也是登场次数最多的一种形态。
    在《最终圣战》中，则设定是由黑暗巨人卡蜜拉、达拉姆、希特拉的部分黑暗力量，以及迪迦本身的黑暗力量所转化的光之形态。

    • 飞行速度：5马赫
    • 奔跑速度：1.5马赫
    • 水中速度：1.5马赫
    • 潜地速度：1.5马赫
    • 跳跃力：800米
    • 握力：50000吨

  2. 空中型（スカイタイプ）

    3. 迪迦的敏捷型态，紫色为主色及以跳跃和速度为主，是身手敏捷的战士。
    在TV版一共出场约为11次，同时也是历代首次登场不以红色为主调的战士型态。
    能在空中进行高速飞行，所以非常擅长空战。但该型态不适合对付力量型的对手以及不擅于应付复数敌人，所以实际使用这形态的局面不多。
    在《最终圣战》中，则设定是吸收了黑暗巨人希特拉的力量所形成的形态。

    • 飞行速度：8马赫
    • 奔跑速度：2马赫
    • 水中速度：2马赫
    • 潜地速度：2马赫
    • 跳跃力：1200米
    • 握力：30000吨

  3. 力量型（パワータイプ）

    4. 迪迦的强力型态，以红色为主色，以力量和爆发力为主的强力战士。
    在TV版共出现约22次，是三形态不借助任何力量时攻击力最强大的形态。拥有着繁多的投技，摔技，威力不小的光线技。
    能用难以言表的怪力与怪兽对战，但是由于速度减慢，对付身手敏捷的对手时则会陷入苦战。
    在《最终圣战》中，则设定是吸收了黑暗巨人达拉姆的力量所形成的形态。

    • 飞行速度：3马赫
    • 奔跑速度：1马赫
    • 水中速度：1马赫
    • 潜地速度：1马赫
    • 跳跃力：500米
    • 握力：80000吨，全力120000吨

  4. 闪耀迪迦（グリッターティガ）

    5. 迪迦的奇迹形态，虽然外型与复合型无异，但一开始变身成该型态时会发出一道金色光芒（光芒则是来自全世界相信迪迦的孩子们心中的光）。
    具有无法估算的攻击力和防御力，曾在剧中连续使出多个必杀绝技并使用最终的必杀技打败了黑暗支配者加坦杰厄以及笼罩在地球上的黑雾。
    在电影版《最终圣战》中，败于暗黑超魔神迪莫杰厄后由无数超古代战士石像所散发的光芒融入死亡的迪迦体内，最终再次登场此形态，
    拥有无法抵御并压制一切的攻击力且免疫一切光线技能的能力，可以使出强大的闪耀哉拉提斯光线”，剧中表现击破888米-无限大的迪莫杰厄的余波击沉了超古代城市露露耶。

    • 飞行速度：无法计算
    • 奔跑速度：无法计算
    • 水中速度：无法计算
    • 潜地速度：无法计算
    • 跳跃力：无法计算
    • 握力：无法计算

故事大纲

       三千万年前的人类已经存在，并发展拥有高度科技的超古代文明，而“迪迦·奥特曼”则为当时造访地球的光之巨人。
但在经过了人类的光暗战争之后，超古代人类自己选择了灭亡的道路，迪迦等其他光之巨人尊重人类的选择，身体则变成了石像形态封于金字塔内，而灵魂则化成光离开地球。
2007年，人类迎来了新的时代，地球不再有侵略和战争，期盼已久的和平终于到来了，但随着怪兽哥尔赞（日语：ゴルザ）出现，其后一颗陨石堕落地球，陨石中的居然是一个立体投影装置，
装置所投影出的人自称是超古代时期的地球警备队队长、人类祭司幽怜，并预言“大地的怪兽”哥尔赞及“天空的怪兽”美尔巴会出现，果然其后马上出现。
而预言中说，可以消灭这两只怪兽的只有迪迦。 而GUTS的新队员“圆大梧”因继承了超古代人的遗传基因，
被幽怜称为“因为你既是光，也是人类。”，成为化身迪迦·奥特曼的人选。

背景设定

年代及世界局势

    本作的时间线设定于2007年至2010年之间，核武器和环境污染已被完全消除及禁止使用的状态。日本东京在剧中被称为“大都市”。
    电影版《最终圣战》则是发生在2012年（即TV版结局2年后）。
    续作《戴拿·奥特曼》则是发生在2017年（即TV版结局7年后）。

世界观背景

    本作所处的世界观，是有着宇宙人、怪兽和光之巨人所活跃的宇宙。
    在显生宙的白垩纪（1.45亿~6600万年前）中，一个名为那加的宇宙文明也从地球带走了多头恐龙，并将其改造成怪兽和恐龙人类，这种物种的化石被人类发现后，被称为新恐龙（新恐竜）。
    在3000万年前的渐新世中，由超古代人所率领的文明开始兴起，并建立无数繁荣的都市，然而过去曾来到地球的“黑暗之王”与祂的眷属，在陨石袭击后从沈睡中苏醒，并派出黑暗派生出以哥尔赞为主的怪兽攻击人类，这时，从猎户座来到地球的“光”，则选中人类中尚未丧失勇气的战士一体化，变成了“光之巨人”，而化身成光之巨人的超古代战士则开始打倒怪兽，将黑暗力量的眷属们封印入神殿深处。
    然而当“邪神卡达诺佐亚”利用“齐杰拉”使当时的人类都沉溺在快乐的梦境世界里，让人类陷入了没有了战争，和平的生活着的幻觉世界，光之巨人因无法干涉人类的选择，因此将自己的身体隐藏在巨大的金字塔中，还原成光的形态离开了地球。最终邪神卡达诺佐亚带着毁灭的黑暗降临，将超古代文明彻底毁灭，幸存的超古代人则搭著飞船离开了地球（详第45话）。
    在电影版《最终圣战》中，则是更注重描述因人们心中的“黑暗”，使得曾和巨人同化的战士成为了黑暗巨人，并开始摧毁文明的过程，在系列中共曾出现7名具有鲜明特征的超古代战士，而圆大梧、柾木庆悟和入麻惠美三人，则个别继承超古代人和超古代地球防卫军队长幽怜的基因。

时间囊

    幽怜在当初文明即将灭亡时，为了告知后世地球的人类关于怪兽和光之巨人的存在，所置放在地球轨道上的超古代器具。
    设定将于地球的文明再次遭遇毁灭的危险时被开启，在电影版《最终圣战》中则是被置放在T.P.C的仓库。

迪迦之地

    迪迦·奥特曼和另外两个残存的巨人在还原成光的形态离开地球前。将躯体隐藏日本东北山区某处的巨大的金字塔中。
    在第一话时遭受哥尔赞及美尔巴的袭击，造成三个巨人中的两位因此完全毁灭，而残存的巨人石像碎片则被采集。
    并以生化研究所的丹后雄二博士为主任的研究团队开始对其进行研究，在第43话曾在熊本市地下洞窟发现“另一个巨人（邪恶迪迦）”的石像遗迹。

超古代遗迹

    三千万年前超古代文明时，人类所建造出的巨型城市遗迹。
    已知遗迹主要为新西兰附近南太平洋海域的‘拉莱耶（ルルイエ/R'lyeh）’，原型取自于克苏鲁神话中的拉莱耶，过去因地摇变动大陆沉没进海洋之中，并在第51话邪神加坦杰厄的复活下而再度从海上浮出。
    除了是邪神卡达诺佐亚的沉睡地点外，也是过去四位邪恶战士及其他多数超古代战士决斗的古战场。

飞船坠落事件

    本传24年前（1985年），有不明飞行物在亚洲坠落，船内的未确认生命体“睡美人（戴西斯星系人）”被捕获，冷冻保存于亚洲某国的旧防卫军基地中。
    8年后保管着飞船残骸以及假死状态的“睡美人”的亚洲联合基地，则受到不明飞行物的大规模袭击，当大量报导那是地外智慧生命体的攻击后
    联合国内部和讨论都变得活跃起来。更加速“Forum2000”的计划和T.P.C的成立。
